Preguntas Lab1:

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es la red global de computadoras interconectadas. Es decir, la infraestructura. Mientras que WWW es un sistema de información para proporcionar acceso a la información como paginas web.


¿Cuáles son las partes de un URL?

Protocol: EL protocolo de comunicación, este genera un request a un servidor que responde con algún resource. Ejemplos: HTTP, HTTPS (versión segura), FTP (para transferencia de archivos entre clientes y un servidor en una red), SMTP (transmisión por mail).

Domain: El nombre del dominio, es decir, la dirección de la página web. Ejemplo: www.google.com

Path: La ruta del recurso que se quiere obtener. Ejemplo: /search?q=hola

Query: Los parámetros que se le pasan al servidor. Ejemplo: ?q=hola


¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET = Leer

HEAD = Similar al GET, peror devuelve los encabezados

PUT = Actualizar

PATCH = Actualizar parcialmente

POST = Subir

DELETE = Eliminar


¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Se debe utilizar el método POST, ya que este método envía los datos al servidor para que sean procesados.


¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

Se utiliza el método GET, ya que las solicitudes GET solo deben recuperar datos.


Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

Significa que la solicitud fue exitosa. No ocurrió ningún error.


¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Si ya que este error significa que el recurso no fue encontrado en el servidor.


¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Si, ya que este error significa que hubo un error interno en el servidor.


¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

Significa que el atributo ya no es parte de la especificación de HTML. Por ejemplo: font, center, big, strike, tt, frame, dir.


¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML5 tiene un mejor soporte para multimedia, tiene una mejor semántica, tiene una mejor conectividad y una mejor experiencia de usuario.


¿Qué componentes de estructura y estilo tiene una tabla?

Desde el encabezado de columna y fila, celdas, bordes,fondo, orden de filtrado,espaciado, color,etc.


¿Cuáles son los principales controles de una forma HTML5?

Input, select, textarea, button, fieldset, label, legend, datalist, output.


¿Qué tanto soporte HTML5 tiene el navegador que utilizas?

Para Google Chrome, este tiene un soporte de 100% HTML5.


¿Cuál es el ciclo de vida de los sistemas de información?

Se refiere a las etapas del sistema desde su concepción hasta su desactivación.


¿Cuál es el ciclo de desarrollo de sistemas de información?

Se refiere al proceso especifico mediante el cual se desarrolla un sistema. Por ejemplo: el análisis, diseño, implementación, pruebas, etc.



Preguntas Lab3:

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Este puede ser usado para darle prioridad a una regla de estilo y en situaciones donde sea no haya manera de lograr un efecto deseado.


Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Porque esta puede entorpecer la claridad y legibilidad de las palabras si no se selecciona un color conveniente.


Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Depende de la situación, pero generalmente se recomienda usar % para tamaños de letra, px para tamaños de caja y pt para tamaños de impresión, ya que no es común para diseño web.


¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque reduce el tiempo de carga de la página y el tamaño del archivo.

¿Cómo se declara una variable con alcance local dentro de una función?

Usando la palabra reservada var, let o const.


¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
